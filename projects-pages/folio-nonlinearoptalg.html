Nonlinear Optimization Algorithms

Optimization/Algorithms Project

MATLAB implementations of the following nonlinear programming algorithms:

  • line-search (simple Wolfe, strong Wolfe, Moré-Thuente),
  • steepest descent,
  • Newton's method,
  • Dogleg method,
  • Steihaug-Toint conjugate gradient trust region method,
  • BFGS,
  • limited memory BFGS,
  • Gauss Newton method.

For the source code, check out the GitHub Repository.
