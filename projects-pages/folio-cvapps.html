Computer Vision Project

MATLAB implementations of the following computer vision applications:

  • 2D object recognition,
  • image mosaicking,
  • line-finders (Hough Transform),
  • an optical flow system,
  • a refocusing app,
  • object reconstruction (photometric stereo, Frankot-Chellappa Algorithm),
  • an object tracking vision system.

For the source code, check out the GitHub Repository.
