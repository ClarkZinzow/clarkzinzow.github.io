Computer Vision Project

MATLAB implementations of the following computer vision applications:

  • 2D object recognition (GitHub),
  • image mosaicking (GitHub),
  • line-finders (Hough Transform) (GitHub),
  • an optical flow system (GitHub),
  • a refocusing app (GitHub),
  • object reconstruction (photometric stereo, Frankot-Chellappa Algorithm) (GitHub),
  • an object tracking vision system (GitHub).
